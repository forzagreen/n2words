# Code of Conduct

## Our Commitment

n2words is committed to providing a welcoming, inclusive, and professional environment for all contributors and users. We value diverse perspectives and strive to build a collaborative community where everyone can contribute meaningfully.

## Expected Behavior

We expect all participants to:

- **Be respectful**: Treat everyone with respect and consideration, regardless of background or experience level
- **Be collaborative**: Work constructively with others, assume good intentions, and help build solutions
- **Be professional**: Keep discussions focused, technical, and productive
- **Be patient**: Help newcomers learn, be understanding of mistakes, and provide constructive feedback
- **Give credit**: Acknowledge others' contributions and build upon their work appropriately
- **Be inclusive**: Use welcoming language and create an environment where everyone feels comfortable participating

## Unacceptable Behavior

The following behaviors are unacceptable:

- Harassment, discrimination, or offensive comments related to personal characteristics (gender, sexual orientation, disability, ethnicity, religion, age, etc.)
- Personal attacks, insults, or trolling
- Publishing others' private information without permission
- Spam, off-topic discussions, or deliberate disruption
- Any conduct that would be inappropriate in a professional setting

## Scope

This Code of Conduct applies to:

- GitHub issues, pull requests, and discussions
- Code review comments
- Documentation and code contributions
- Any communication in project spaces

It also applies when representing the project in public spaces.

## Enforcement

### Reporting

If you experience or witness unacceptable behavior, please report it:

1. **GitHub**: Open a confidential issue or contact maintainers directly
2. **Email**: security@vigario.tech (for serious violations)

All reports will be reviewed confidentially.

### Enforcement Actions

Violations may result in:

1. **Warning**: Private notification about the violation
2. **Temporary Ban**: Temporary restriction from project participation
3. **Permanent Ban**: Permanent removal from the project

Actions will be proportional to the severity and frequency of violations.

### Appeal Process

If you believe an enforcement action was made in error, you may appeal by contacting the project maintainers with additional context.

## Attribution

This Code of Conduct is adapted from industry best practices and tailored for the n2words project's focus on technical collaboration and inclusive contribution.

## Questions?

For questions about this Code of Conduct, please:

- Open a discussion on [GitHub Discussions](https://github.com/forzagreen/n2words/discussions)
- Contact project maintainers
